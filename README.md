# CFC — Compressed Folder Creator  
  
**Создавай ZIP-архивы прямо в браузере, без установки, без ограничений.**  

> CFC — это **файловый менеджер + браузерный компрессор.** Никаких данных на сервер — **всё у тебя в памяти и в итоговой папке**

## Возможности

- 📂 **Виртуальный файловый менеджер** — создавай папки, загружай файлы, переходи внутрь.
- 🗜️ **Компрессия в ZIP** — любой размер файлов (сколько потянет браузер).
- 💾 **Работает без интернета** — после загрузки страницы не требует сети.
- 🔒 **100% локально** — данные не покидают твой компьютер.

---

## 🚀 Быстрый старт

1. Скачай `index.html`.
2. Открой в любом браузере (Chrome, Firefox, Edge, Safari).
3. Нажми **«+» → Файл** и выбери файлы с диска.
4. Нажми **«+» → Папка**, создай структуру.
5. Кликай на папки — перемещайся внутри.
6. Нажми **«Экспорт»** → скачается ZIP со всеми файлами и папками.

> 💡 **Совет:** Используй CFC как быстрый компрессор для флешки — всегда под рукой, без установки.
